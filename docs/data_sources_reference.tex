\textbf{LCOH Formula:}
\begin{equation}
\text{LCOH} = \frac{\text{CAPEX} \times \text{CRF} + \text{Fixed OPEX} + \text{Stack Replacement}}{\text{Annual H}_2\text{ Production}} + \text{Electricity Cost}
\end{equation}

Where CRF = Capital Recovery Factor (8\% discount rate, 20-year lifetime)
