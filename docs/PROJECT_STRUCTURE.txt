Korean Petrochemical MACC Model (Release Layout)
================================================

Root:
├── README.md                       # GitHub-ready overview
├── MODEL_DOCUMENTATION.md          # Final methodology and assumptions
├── LICENSE                         # MIT license
├── requirements.txt                # Python dependencies
├── run_all_scenarios_v3.py         # Execute 3 production × 2 tech scenarios
├── generate_professional_figures.py# Recreate publication figures
├── data/                           # Input assumptions (CSV/XLSX)
├── modules/                        # Core model code
└── outputs/figures/                # Pre-generated figures for reference

Key data files (data/):
- facility_database.csv             # 248 facilities with capacity + product
- energy_intensities.csv            # Process energy intensities
- emission_factors.csv              # Fuel emission factors
- fuel_price_trajectory.csv         # Fossil fuel price path
- h2_price_trajectory.csv           # Hydrogen price path
- re_price_trajectory.csv           # Renewable price path
- grid_emission_trajectory.csv      # Grid emission factors
- demand_growth_trajectory*.csv     # Production pathways (Shaheen, restructuring)
- technology_parameters.csv         # CAPEX/OPEX, efficiencies, availability

Core modules (modules/):
- baseline.py                       # Baseline emissions + BAU trajectory
- macc.py                           # Energy-based MACC calculation
- optimization_v2.py                # Cost optimization (mutually exclusive NCC tech)
- data_manager.py                   # Centralized data loading/validation
- utils.py                          # Shared helpers (plots, loaders, calculators)

Outputs:
- outputs/figures/figure*.png       # Seven publication-quality figures
- Running the model will recreate module_* outputs under outputs/
