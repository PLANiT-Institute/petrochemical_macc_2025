# Data Availability Statement

All data and code required to reproduce the manuscript's quantitative claims are included in the submission package.

- Core data: `03_data/`
- Reproducibility scripts: `04_reproducibility/`
- Verification pipeline and outputs: `06_verification/`

Claim-level traceability is provided via `06_verification/claim_register.csv` and `06_verification/claim_checks.csv`.
