## 🎯 시나리오 구성 (Main vs Reference)

### Main Scenario (정책 선호)
- **기술**: NCC-전기로 (NCC-Electricity)
- **특징**: 전기화 기반, 재생에너지 전력 사용
- **비용**: $76/tCO2 (2050)
- **장점**: 장기 전력망 탈탄소와 시너지

### Reference Scenario (경제 최적)
- **기술**: NCC-수소 (NCC-H2)
- **특징**: 그린수소 연소
- **비용**: $69/tCO2 (2050) - **9% 저렴**
- **장점**: 현재 비용 가정에서 경제적 우위

**⚠️ 중요 노트**: 현재 최적화는 경제성 기준으로 NCC-H2를 자동 선택합니다. NCC-전기로를 강제하려면 별도 정책 시나리오 설정이 필요합니다.
