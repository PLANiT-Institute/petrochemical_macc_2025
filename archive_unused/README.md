# 아카이브 개요

이 디렉터리는 현재 에너지 기반 MACC 모델에서 사용하지 않는 레거시 자료를 보관합니다.  
필요 시 참고용으로만 유지하며, 모델 실행에는 영향을 주지 않습니다.

## 포함 항목

- `legacy_lcoe_approach/`  
  - 이전 LCOE 기반 구현 전체 (스크립트, 보고서, 참고 자료 등 30여 개 문서)
  - 데이터 예시: `ncc_lcoe_trajectory.csv`, `macc_lcoe_version.py`
- `legacy_output/`  
  - 2024년 버전에서 생성된 옛 MACC 그래프와 CSV (`macc_curve_2030_final.png` 등)
  - 동일한 최신 결과물은 `outputs/module_02/`에 재생성됨
- `scripts/run_module_02_refactored.py`  
  - 리팩터링 이전 구조를 유지하던 실행 스크립트

## 사용 시 주의사항

1. 최신 결과는 `outputs/module_*/` 경로를 사용하세요.
2. 레거시 자료가 필요하면 해당 파일을 복사하여 비교 분석용으로만 활용하세요.
3. 아카이브 내 파일을 수정하거나 실행해도 현재 모델 결과에는 반영되지 않습니다.

필요에 따라 추가로 옛 파일을 옮길 때는, 이 문서를 업데이트하고 이동 사유를 간단히 기록해주세요.
